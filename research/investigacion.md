## Diferencia entre Página Web y Aplicación Web

**Página web:**  
Sitio informativo con contenido estático o limitado, enfocado en mostrar información.

**Aplicación web:**  
Sistema interactivo que permite al usuario realizar acciones, procesar datos y guardar información.

Si tu proyecto solo necesita brindar información básica y estática, como una página de inicio o un sitio web de una empresa, una página web podría ser la mejor opción ya que es más accesible, menos costosa y más fácil de desarrollar que una aplicación web. Por otro lado, si necesitas una plataforma más interactiva, con funcionalidades avanzadas y una experiencia de usuario personalizada, una aplicación web podría ser la mejor opción.

## Arquitectura general de una aplicación web

- **Frontend:** Interfaz de usuario (HTML, CSS, JavaScript)
- **Backend:** Lógica de negocio y APIs
- **Base de datos:** Almacenamiento persistente
- **Entornos:** Desarrollo, pruebas y producción
