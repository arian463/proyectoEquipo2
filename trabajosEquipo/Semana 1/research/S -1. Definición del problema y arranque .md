## Investigación

Página web

Un sitio web reúne varias páginas bajo un mismo dominio, conectadas entre sí. Su función principal es informar, mostrando contenido estático o de solo lectura, su creación es más sencilla y rápida.

Aplicación web

Una aplicación web es un programa en línea que no requiere descarga. Se centra en la interactividad, permitiendo al usuario realizar tareas y enviar datos que el sistema procesa para dar respuestas. Ejemplos: Gmail, YouTube, Amazon. A diferencia de las páginas web públicas, las aplicaciones suelen pedir registro de usuario para acceder a funciones personalizadas.

## Problemas que resuelven con software  

### Lentitud del sistema

El primer problema que muchas veces encontramos es la lentitud del sistema, que puede ser causada por una variedad de factores, incluyendo la acumulación de archivos temporales, tener demasiados programas ejecutándose simultáneamente o incluso la presencia de software malicioso. La solución a este problema incluye limpiar regularmente el sistema con herramientas de optimización, cerrar aplicaciones en segundo plano que no se estén utilizando, y realizar análisis periódicos en busca de malware.

### Conflictos del software

Otro problema común son los conflictos de software, que ocurren cuando distintas aplicaciones o el software y los drivers entran en conflicto, provocando mal funcionamiento o que ciertas aplicaciones dejen de funcionar completamente. Para solucionar estos conflictos, es vital mantener todo el software y los drivers actualizados. Si se detecta un conflicto específico, una opción es intentar restaurar el sistema a un punto anterior a la instalación del software problemático o consultar con soporte técnico para una solución específica.

### Falta de compatibilidad

La falta de compatibilidad es otro desafío frecuente, especialmente cuando el software antiguo no funciona correctamente en sistemas operativos más nuevos. Las soluciones incluyen buscar actualizaciones del software que mejoren la compatibilidad o utilizar modos de compatibilidad dentro del sistema operativo. Si estas opciones no están disponibles, se puede considerar el uso de una máquina virtual para ejecutar una versión del sistema operativo compatible con el software antiguo.

### Fallos de seguridad

Los fallos de seguridad no solo ponen en riesgo nuestros datos personales, sino que también pueden afectar el rendimiento del sistema. Mantener el software actualizado y utilizar herramientas de seguridad confiables son pasos fundamentales para solucionar y prevenir estos problemas. Es vital realizar análisis periódicos con software antivirus y mantenerse informado sobre las últimas amenazas de seguridad.

### Pérdida de datos

La pérdida de datos puede ser devastadora, ya sea por errores del sistema, fallos de hardware o ataques de malware. La implementación de una estrategia sólida de respaldo y recuperación de datos es esencial para mitigar este problema. Asegúrate de realizar copias de seguridad regulares de tus datos importantes y considera el uso de soluciones de almacenamiento en la nube para una capa adicional de seguridad.  
Además de estos problemas específicos, es crucial mantener tu sistema y aplicaciones actualizados, ya que las actualizaciones no solo traen nuevas características, sino que también corrigen errores y mejoran la seguridad.
