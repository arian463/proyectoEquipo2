---Diferencia aplicacion web y pagina web

Página web

Informativa

Contenido estático

No hay usuarios ni lógica compleja

Ejemplo: página de una empresa, blog, portafolio

Aplicación web

Interactiva

Tiene usuarios, roles, datos

Frontend , Backend y Base de datos

Ejemplo: Netflix, Uber, Airbnb, Spotify

---Analizar 2 plataformas reales similares a tu idea

Plataforma 1: Calendly

Calendly es una aplicación web que permite a profesionales y empresas gestionar citas de forma automática, sincronizando la disponibilidad del usuario con un calendario y permitiendo que los clientes reserven horarios disponibles sin interacción directa.

Usuarios objetivo

Profesionales independientes

Empresas de servicios

Equipos de trabajo que requieren reuniones programadas

Funcionalidades principales

Configuración de horarios disponibles

Reservas automáticas 24/7

Confirmaciones y recordatorios por correo

Integración con calendarios externos (Google Calendar, Outlook)

Problema que resuelve

Elimina la necesidad de coordinar citas manualmente mediante correos o mensajes, evitando confusiones y pérdida de tiempo tanto para el cliente como para el proveedor del servicio.

Relación con nuestro proyecto

Nuestra aplicación comparte el mismo objetivo principal: permitir que los clientes agenden citas en tiempo real. Sin embargo, nuestro sistema se enfocará en establecimientos físicos con múltiples servicios y empleados, mientras que Calendly está más orientado a reuniones profesionales.

Plataforma 2: Setmore

Setmore es una plataforma de gestión de citas dirigida a pequeños y medianos negocios que ofrecen servicios presenciales. Permite administrar horarios, empleados, servicios y reservas desde una aplicación web centralizada.

Usuarios objetivo

Consultorios médicos

Centros de estética

Talleres y negocios de servicios

Pequeñas empresas con atención al público

Funcionalidades principales

Gestión de servicios y duración

Administración de empleados

Calendario de citas centralizado

Reservas en línea para clientes

Problema que resuelve

Ayuda a los negocios a reducir errores humanos en la agenda, mejorar la organización interna y ofrecer una mejor experiencia al cliente mediante reservas en línea.

Relación con nuestro proyecto

Setmore es una referencia directa para nuestro sistema, ya que maneja múltiples servicios y empleados. Nuestra aplicación busca implementar un alcance inicial similar, pero de manera más simple y adaptable a distintos tipos de establecimientos.
